Variable Keywords in JavaScript:-

- Variable keywords are used to declare a variable and to initialize a value to  the variables.
- There are Three(3) types of Variable in js var,let,const,
  VAR:-
  - if a  variable is declare using var keyword we can re declare and reinitialize the same variable.
  - Var is global scope variable
  LET:-
  - if a  variable is declare using let keyword we can not re-declare the variable but we can  reinitialize the variable.
  - let is a block scpoe variable
  CONST:-
  - const stands for constant
  - if a  variable is declare using cont keyword we can neither re-declare  nor reinitialize the variable.
  - Const is a block scope variable
